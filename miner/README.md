# HashLatch GPU Miner

## Quick Start

1. Download SRBMiner from https://github.com/doktor83/SRBMiner-Multi/releases
2. Extract to this folder
3. Run:
   - **Windows:** Double-click `start.bat`
   - **Linux:** `chmod +x start.sh && ./start.sh`
4. Enter your HLC address when prompted
5. Start mining!

## Requirements

- NVIDIA GPU (GTX 10xx or newer) or AMD GPU (RX 4xx or newer)
- At least 4GB VRAM
- Windows 10/11 or Linux (Ubuntu 20.04+)
